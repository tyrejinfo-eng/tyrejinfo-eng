# 📌 Before You Ask for Help

Please make sure you:

* Read the relevant documentation
* Check existing GitHub Issues
* Search Discussions
* Verify you are using the latest version

# 📋 How to Ask Good Questions

When requesting support, include:

* Repository name
* Version
* Operating system
* Steps to reproduce the issue
* Expected behavior
* Actual behavior
* Logs or screenshots (if available)

---

# ⚙ Common Areas of Support

We commonly assist with:

* Build issues (.NET / solution setup)
* Worker execution problems
* Orchestrator behavior
* AI provider configuration
* Model loading issues
* Plugin integration
* Runtime errors
* Performance debugging

---

# 🚫 What Support Does NOT Cover

* Custom feature development on request
* Unrelated third-party software issues
* Non-reproducible vague bugs
* General programming tutorials outside the ecosystem
